# perDM MVP Prototype

Clickable SaaS prototype for perDM, focused on business trip expense documentation, receipt verification, and report export for accountants and finance teams.

## Open locally

Open `outputs/perdm-mvp/index.html` in a browser.

## Demo flow

1. Open the demo dashboard.
2. Open Nino Beridze's Batumi business trip.
3. Review card and cash expenses.
4. Check missing receipts.
5. Attach a receipt.
6. Generate the report preview.
7. Use the PDF / Excel download buttons as mocked actions.
